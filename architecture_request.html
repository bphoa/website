If you would like to make modifications to your house or property, you will need to submit an Architecture Request Form. Some example modifications include:

  • Painting
  • Roofing
  • Fencing
  • Screened Porch
  • Swimming Pool / Spa
  • Garage Door / Front Door / Windows
  • Skylights / Solar Panels
  • Landscaping
